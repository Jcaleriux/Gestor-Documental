Proyecto Novogar

Seguridad y control actual

Proyecto Novogar ya incorpora autenticación, permisos, trazabilidad y control de acceso

08 / 15

Controles ya presentes

  • Login autenticado con JWT para sesión de usuario.
  • Permisos por rol para ver, comentar, contabilizar, tramitar o aprobar.
  • Acceso por sociedad para limitar alcance operativo según asignación.
  • Protección de archivos: PDF/XML requieren autenticación y permiso para visualizar o descargar.
  • Rate limit en login y headers básicos de seguridad en backend.

Trazabilidad y evidencia

  • Historial de estados y movimientos.
  • Auditoría con actor responsable y fecha.
  • Comentarios asociados al documento o trámite.
  • Separación entre estado documental y workflow de pago.

Capacidades que se pueden incorporar

  • Segundo factor de autenticación para accesos sensibles.
  • Mayor integración con servicios corporativos de identidad y gestión de secretos.
  • Alertas y monitoreo operativo más visibles para soporte y seguimiento.

Qué significa esto

Proyecto Novogar ya dispone de controles de acceso y trazabilidad para operar internamente. El siguiente paso es llevar esos controles a un entorno corporativo con secretos, red, respaldo y continuidad formalizados.
