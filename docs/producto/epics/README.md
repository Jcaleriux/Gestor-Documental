# Epics

## Objetivo

Cada epic describe una iniciativa grande que puede abarcar varias historias o varios sprints.

## Convencion sugerida

- `EPIC-A-*`: epics activos o propuestos
- `EPIC-H-*`: epics historicos ya cerrados

## Regla practica

Si un trabajo:

- toca varios modulos,
- dura mas de un sprint,
- o necesita decisiones de alcance y riesgo,

entonces merece epic.
