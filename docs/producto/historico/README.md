# Historico Relevante

## Objetivo

Este directorio no busca reconstruir todo el pasado del proyecto. Solo conserva contexto historico que aun ayuda a entender decisiones actuales.

## Que si conviene guardar

- epics historicos cerrados que cambiaron arquitectura
- hitos de deuda tecnica importante ya resuelta
- decisiones que explican por que el sistema hoy funciona como funciona

## Que no conviene guardar aqui

- cada tarea pequena vieja
- cada bug puntual cerrado
- historico demasiado detallado que ya no ayuda a decidir nada
