# Mermaid Sources

Archivos Mermaid versionados para arquitectura y ERD.

## Archivos
1. `arquitectura_contexto.mmd`: diagrama de contextos y dependencias.
2. `erd_objetivo.mmd`: relaciones clave del modelo de datos objetivo.

## Uso en Draw.io
1. Abrir `app.diagrams.net`.
2. `Insert` -> `Advanced` -> `Mermaid`.
3. Pegar contenido del archivo `.mmd`.

## Uso en Markdown
Puedes embeberlos con bloque:

```mermaid
# contenido del .mmd
```
